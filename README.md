# 中国革命历史学习网站

一个交互式的中国革命历史学习网站，包含时间轴、知识图谱和测验功能。

## 项目特点

- 📚 **丰富的历史内容** - 涵盖从鸦片战争到中华人民共和国成立的重要历史事件
- 🎯 **交互式知识图谱** - 使用D3.js构建的可视化知识网络
- 📱 **响应式设计** - 完美适配手机、平板和电脑
- 🧠 **智能测验系统** - 包含10道选择题的知识测试
- ⏱️ **时间轴筛选** - 按历史时期筛选查看事件

## 技术栈

- **前端**: HTML5, CSS3, JavaScript (ES6+)
- **可视化**: D3.js (v7)
- **样式**: CSS Grid, Flexbox, CSS变量
- **响应式**: 移动端优先设计

## 主要功能

### 1. 知识图谱
- **36个节点** (21个历史事件 + 15个重要人物)
- **44个连接关系** 展示历史关联
- **交互功能**: 点击查看详情、拖拽节点、缩放和平移
- **优化显示**: 支持完全展开，适配各种屏幕尺寸

### 2. 历史时间轴
- 按时期筛选: 全部、旧民主主义革命、新民主主义革命
- 平滑动画效果
- 详细的事件描述

### 3. 知识测验
- 10道选择题，每题10分
- 实时进度显示
- 成绩评估和反馈

## 修复和优化

### 已解决的问题
1. ✅ **GitHub Pages显示问题** - 修复了CSS和JavaScript文件路径引用
2. ✅ **移动端显示问题** - 添加了完整的响应式设计
3. ✅ **知识图谱无法完全展开** - 优化了D3.js力导向图参数

### 知识图谱优化详情
- **增加节点间距**: `distance: 150`
- **增强排斥力**: `strength: -400`
- **添加碰撞检测**: `forceCollide().radius(60)`
- **减慢布局衰减**: `alphaDecay: 0.02`
- **添加缩放功能**: 支持0.1-4倍缩放
- **viewBox支持**: 确保在不同设备上正确显示

## 部署到GitHub Pages

1. 将所有文件推送到GitHub仓库
2. 在仓库设置中启用GitHub Pages
3. 选择部署分支（通常是main或master）
4. 访问 `https://[你的用户名].github.io/[仓库名]`

请参考 `DEPLOYMENT.md` 文件获取完整的部署说明。
