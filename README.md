# 中国革命历史时间轴 (1840-1949)

一个交互式的中国革命历史时间轴网页项目，展示1840年至1949年中国革命历史的重要事件、人物和知识图谱。

## 项目特点

- 📅 **交互式时间轴** - 展示中国革命历史的重要事件
- 🎯 **知识图谱** - 可视化历史事件与人物之间的关系
- 🧠 **趣味测验** - 测试对中国革命历史的了解程度
- 📱 **响应式设计** - 适配各种设备屏幕
- 🎨 **中国风设计** - 采用中国红和金色主题色彩

## 文件结构

```
├── index.html          # 主页面文件
├── style.css           # 样式文件
├── main.js             # 主要JavaScript逻辑
├── data.js             # 测验题目和知识图谱数据
└── README.md           # 项目说明文档
```

## 部署到GitHub Pages

### 方法一：直接上传文件

1. 在GitHub上创建一个新的仓库
2. 将所有文件上传到仓库的根目录
3. 在仓库设置中启用GitHub Pages：
   - 进入仓库的 **Settings** 页面
   - 在左侧菜单中找到 **Pages**
   - 在 **Source** 部分选择 **Deploy from a branch**
   - 选择 **main** 分支和 **/(root)** 文件夹
   - 点击 **Save**

### 方法二：使用Git命令

```bash
# 初始化Git仓库
git init

# 添加所有文件
git add .

# 提交更改
git commit -m "Initial commit: 中国革命历史时间轴项目"

# 连接到GitHub仓库
git remote add origin https://github.com/你的用户名/你的仓库名.git

# 推送到GitHub
git branch -M main
git push -u origin main
```

## 访问您的网站

部署完成后，您的网站将在以下地址访问：


## 功能说明

### 时间轴功能
- 按时期筛选历史事件
- 平滑滚动导航
- 详细的事件描述和关键人物信息

### 知识图谱
- 使用D3.js创建的可视化图谱
- 展示历史事件与人物之间的复杂关系
- 交互式节点点击查看详细信息

### 趣味测验
- 10道选择题测试历史知识
- 实时得分和结果反馈
- 可重复测试

## 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式和动画
- **JavaScript** - 交互逻辑
- **D3.js** - 知识图谱可视化
